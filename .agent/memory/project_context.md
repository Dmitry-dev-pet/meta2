# Контекст Проекта

## Обзор
**Data Importer Service** — это микросервис на FastAPI, предназначенный для импорта и обработки данных из Google Sheets в базу данных платформы сообщества IT-менторов. Он служит критически важным мостом между ручным вводом данных (Google Sheets) и структурированным хранилищем данных платформы.

## Основная Цель
Сервис обеспечивает:
1.  **Качество данных**: В систему попадают только валидные и отфильтрованные данные.
2.  **Источник аутентификации**: Пользователи импортируются только при наличии Telegram-аккаунта, что позволяет последующий вход через Telegram Mini App.
3.  **Связывание данных**: Проекты корректно привязываются к студентам (через GitHub), а ревью — к менторам и проектам.

## Ключевые Возможности
-   **Стратегия полной замены**: Полное обновление данных при каждом импорте для обеспечения согласованности.
-   **Умная фильтрация**:
    -   **Студенты**: Должны иметь Telegram username ИЛИ GitHub URL.
    -   **Менторы**: Должны иметь Telegram username.
-   **Асинхронная обработка**: Неблокирующий импорт с отслеживанием статуса.
-   **Поддержка двух БД**: SQLite для разработки, PostgreSQL для продакшена.

## Архитектура
Проект следует **Слоистой Архитектуре**:
-   **API Layer**: Эндпоинты FastAPI для управления.
-   **Service Layer**: Бизнес-логика (`ImportService`, `DataProcessor`).
-   **Data Access Layer**: SQLAlchemy ORM и Pydantic модели.
-   **Infrastructure**: Интеграция с Google Sheets, Логирование, Конфигурация.

## Текущее Состояние
-   **Статус**: Полностью реализован и верифицирован.
-   **Верификация**: Сервер запускается корректно, тестовый импорт (dry-run) проходит успешно.
-   **Следующие шаги**: Конфигурация для продакшн-развертывания, интеграция с основной платформой.
