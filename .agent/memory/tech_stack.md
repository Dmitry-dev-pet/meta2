# Технологический Стек

## Ядро
-   **Язык**: Python 3.13+
-   **Фреймворк**: FastAPI 0.121.1
-   **Сервер**: Uvicorn

## Данные
-   **ORM**: SQLAlchemy 2.0.44 (Async)
-   **Валидация**: Pydantic v2
-   **Миграции**: Alembic
-   **Базы Данных**:
    -   SQLite (Разработка)
    -   PostgreSQL (Продакшн)
    -   Драйверы: `aiosqlite`, `asyncpg`

## Внешние Сервисы
-   **Google Sheets API v4**: Источник данных
-   **Google Auth**: Аутентификация через Service Account

## Инструменты Разработки
-   **Пакетный Менеджер**: Rye
-   **Линтинг/Форматирование**: Ruff, Black
-   **Тестирование**: Pytest
-   **Окружение**: `.env` (python-dotenv)

## Инфраструктура
-   **Контейнеризация**: Docker (подразумевается/планируется)
-   **Скрипты**: Bash скрипты для инициализации, запуска, тестов (`scripts/`)
