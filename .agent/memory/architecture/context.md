# System Context (C1)

## Система
**Data Importer Service** — микросервис для импорта данных из Google Sheets в платформу IT Mentor Community.

## Пользователи
- **Администраторы**: Запускают импорт, мониторят процесс
- **Студенты**: Данные импортируются для последующей аутентификации через Telegram
- **Менторы**: Данные импортируются для связи с ревью и проектами

## Внешние Системы
- **Google Sheets API**: Источник сырых данных
  - Main Spreadsheet: студенты, проекты, ревью
  - Mentors Spreadsheet: менторы
- **Telegram**: Требуется для импорта пользователей (username обязателен)
- **GitHub**: Используется для связи проектов со студентами (по URL репозитория)

## Границы Системы
**Входит в систему:**
- Fetch данных из Google Sheets
- Валидация и фильтрация
- Трансформация данных
- Запись в БД

**Вне системы:**
- Ввод данных в Google Sheets (ручной процесс)
- Telegram-аутентификация (другой сервис)
- Основная платформа менторства

## Ключевые Требования
- **Качество данных**: Только валидные записи попадают в БД
- **Telegram-фильтр**: Пользователи без Telegram username не импортируются
- **Целостность связей**: Ревью корректно привязаны к менторам и проектам
